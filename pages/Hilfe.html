On this page, I show examples of different types of presentation, which should demonstrate how you can make your own page later. Here, images of products (travel) can be combined with associated image texts and a comprehensive main text. The various representations (cards, slider, masonry, single image, product gallery) present identical product contents in different ways.
Display style as well as main text, pictures and picture texts can later be graphically defined in the backend in the MODX version. All texts can be individually formatted with colors, font sizes, underlines, borders etc. Unfortunately, this can not be demonstrated in this pre-release version. If you prefer the Non Modx version without backend, all this can be modified directly in HTML and CSS code.
{{> contentHr}}

Cards

This text has been defined to take all of the available width. It was placed above the image part. You can also choose the text width smaller and put the main text left or right of the image part, see below at Slider! This presentation shows the defined images and their associated texts as cardss that can be clicked to get all the information displayed in an extra window. The number is not limited, for more cards a new row is opened. You can also see that if you reduce the size of your browser window, the display will eventually jump from five to three columns and finally only two.
{{#with products}} {{> contentCards}} {{/with}}
{{> contentHr}}

Slider

This text takes only half the available width. It was placed to the right of the image part. There are more text widths to choose from. You can also insert the main text to the left of the image part. Both will only be displayed side by side in the browser if there is sufficient screen width available, ie on normal computers. On smartphone displays text and image part are always displayed on top of each other for space reasons. Slider shows the pictures as a slide show.
{{#with products}} {{> contentSlider}} {{/with}}

Masonry

With Masonry, images and their associated image descriptions are distributed evenly over several columns. All image descriptions can be formatted. You can see how the layout changes when you zoom in or out of the browser window. Masonry is probably especially suitable for animated content.
{{#with products}} {{> contentMasonry1}} {{/with}}
{{> contentHr}}

Single

With Single, only the very first defined image is displayed, all others are ignored. To see all pictures, you can choose Gallery as presentation, see below! If no image has been defined, only the main text will be displayed. This is a way to put texts here, when using Modx. The texts are entered using a graphical editor and can also be formatted with colors, font sizes, underlines, borders etc. You can also insert other images in any position and in any size in the main text and so make the layout of the page. The text width shrinks when the browser window is resized. Without using Modx, you have direct access to HTML Code and can enter Text as you like. Though, you have to be more careful, of course.

Gallery

Gallery allows to click through all available pictures. The corresponding titles are also displayed, if they exist. However, the text descriptions for each picture, which can be read individually by clicking on Cards, can not be seen here.
